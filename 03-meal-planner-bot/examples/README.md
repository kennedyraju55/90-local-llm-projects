# Examples for Meal Planner Bot

This directory contains example scripts demonstrating how to use this project.

## Quick Demo

```bash
python examples/demo.py
```

## What the Demo Shows

- **`generate_meal_plan()`** — Generate a meal plan based on preferences.
- **`get_recipe_details()`** — Get a detailed recipe for a specific meal.
- **`generate_shopping_list()`** — Generate a consolidated shopping list from a meal plan.

## Prerequisites

- Python 3.10+
- Ollama running with Gemma 4 model
- Project dependencies installed (`pip install -e .`)

## Running

From the project root directory:

```bash
# Install the project in development mode
pip install -e .

# Run the demo
python examples/demo.py
```
