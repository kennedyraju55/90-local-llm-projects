# Examples for Travel Itinerary Bot

This directory contains example scripts demonstrating how to use this project.

## Quick Demo

```bash
python examples/demo.py
```

## What the Demo Shows

- **`generate_itinerary()`** — Generate a travel itinerary.
- **`generate_multi_destination_itinerary()`** — Generate a multi-destination itinerary.
- **`get_place_details()`** — Get detailed information about a specific place or attraction.
- **`generate_budget_breakdown()`** — Generate a budget breakdown for the itinerary.
- **`generate_packing_list()`** — Generate a packing list for the trip.

## Prerequisites

- Python 3.10+
- Ollama running with Gemma 4 model
- Project dependencies installed (`pip install -e .`)

## Running

From the project root directory:

```bash
# Install the project in development mode
pip install -e .

# Run the demo
python examples/demo.py
```
